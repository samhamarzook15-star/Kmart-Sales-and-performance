# 📊 Kmart Sales Dashboard (Fictional Dataset)

## 📌 Overview

A Power BI dashboard analyzing retail sales performance using a fictional Kmart Australia dataset.

The project focuses on sales trends, product performance, and customer behavior insights.

## 🎯 Goal

To build an interactive dashboard that helps analyze:
- Sales performance over time
- Top products, categories, and regions
- Online vs in-store trends
- Key business patterns and changes

## 🛠️ Tools

- Microsoft Excel (Data cleaning & preparation)
- Microsoft Power BI (Dashboard development & visualization)

## ⚙️ Process

- Cleaned and structured the raw dataset in Excel
- Performed exploratory analysis using Pivot Tables
- Built an interactive Power BI dashboard with KPI cards, charts, maps, and slicers

## 📈 Key Insights

- Profit growth (3.20%) outperformed revenue growth (1.78%)
- Growth was driven primarily by higher Average Order Value
- The top-performing category shifted from Clothing to Beauty
- Customer demand shifted from Hoodies to Jackets

## 💡 Recommendations

- Expand investment in the Beauty category
- Increase purchase frequency through promotions and loyalty programs
- Replicate strategies from high-performing stores
- Improve seasonal inventory planning
