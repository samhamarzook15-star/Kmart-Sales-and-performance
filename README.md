📊Kmart Sales Dashboard (Fictional Dataset)

📌 Overview

A Power BI dashboard analyzing retail sales performance using a fictional Kmart Australia dataset.
The project focuses on sales trends, product performance, and customer behavior insights.

🎯 Goal

To build an interactive dashboard that helps analyze:

Sales performance over time
Top products, categories, and regions
Online vs in-store trends
Key business patterns and changes

🛠 Tools

Excel (data cleaning & preparation)
Power BI (dashboard & visualization)

⚙️ Process

Cleaned and structured raw dataset in Excel
Built Pivot-based analysis for exploration
Developed interactive Power BI dashboard with slicers and KPI cards

📈 Key Insights
Profit growth (3.20%) outperformed revenue growth (1.78%)
Growth driven mainly by higher Average Order Value
Category shift from Clothing → Beauty observed
Product demand shifted from Hoodies → Jackets

💡 Recommendations

Focus more on Beauty category expansion
Increase purchase frequency through offers and bundles
Improve seasonal inventory planning
Replicate high-performing store strategies
